Camping Equipment Rental Database

Database Systems, Ottawa, Ontario
Semptember 2024 – December 2024

  • Developed a database management system for handling camping equipment rentals, allowing users to view available equipment, make reservations, and track rental records.
  • Implemented SQL queries for data retrieval and management, ensuring efficient handling of user requests.
  • Designed and maintained the database using MySQL, optimizing the system for real-time data access and updates.
  • Enhanced user interaction by integrating front-end forms with the database for seamless reservation processing.
  • Improved database security by configuring Access controls and managing user permissions for rental transactions.
  • Demonstrated strong knowledge in database management, deepening understanding of data retrieval and system architecture.
  • Built an additional Oracle database to handle complex operations and enhance data integrity.
  • Designed and implemented triggers and stored procedures in Oracle to automate data validations and streamline insert, update, and delete actions.
  • Used MS Access as a front-end interface to connect with the Oracle database, allowing seamless interaction and user-friendly access.
